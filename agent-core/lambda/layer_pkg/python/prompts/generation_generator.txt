Type mapping rules:

- uint8_t / int8_t → "uint8" / "int8"

- uint32_t / int32_t → "uint32" / "int32"
- uint64_t / int64_t → "uint64" / "int64"
- float → "float", double → "double"
- bool / uint8_t used as boolean → "uint8"
- StructType field[N] → {"type": "array", "length": N, "element": "StructType"}
