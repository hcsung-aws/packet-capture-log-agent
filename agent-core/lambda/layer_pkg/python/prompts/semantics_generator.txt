{
  "interaction_sources": [
    {"source_prefix": "SC_OTHER_CHAR", "condition": "SC_OTHER_CHAR.charUid > 0"}
  ],
  "state_conditions": [
    {"action_pattern": "party", "state_field": "SC_PARTY_UPDATE.memberCount", "min_value": 2}

Respond with JSON only.
